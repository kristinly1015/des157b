Arrow Functions

This function returns the names of my pets.

Named Function

             function pets(petname1, petname2, petname3){
                return `My pets are ${petname1}, ${petname2}, and ${petname3}`;
                }

Function Expression

            const pets = function(petname1, petname2, petname3){
                return `My pets are ${petname1}, ${petname2}, and ${petname3}`;
            }

Arrow Function

            const pets = (petname1, petname2, petname3) => `My pets are ${petname1}, ${petname2}, and ${petname3}`;
